# Delivery Exception Management Dashboard

An internal operations tool designed for **GreyAtom Logistics** to log, track, and manage delivery exceptions efficiently.

## Live Demo

The project is deployed using Netlify.

🔗 Live Website:  
https://sprightly-entremet-dd7f2f.netlify.app

## Features

- **Log Exceptions:** Easily submit new exceptions with Delivery ID, Customer Name, Issue Type, Priority, and Notes.
- **Issue Types Supported:** Address Not Found, Customer Not Available, Payment Issue, Package Damaged.
- **Interactive Dashboard:** View all logged exceptions in a clean, tabular format.
- **Filtering System:** Filter exceptions by Issue Type (e.g., Payment issues, Damaged packages) or by Status (Open vs. Resolved).
- **Live Statistics:** Keep track of metrics like *Total Open* and *Total Resolved* issues dynamically.
- **Status Management:** Quickly mark an open exception as 'Resolved' via action buttons.

## Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to Use

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use the form on the left/top to log a new delivery exception.
4. Manage, filter, and track statuses in the Dashboard section below.
